git

2019/7/21 posted in  1 Tools

git 是一个版本管理的工具

1 基本配置

（1）生成公钥
ssh-keygen -t rsa -f ~/.ssh/id_rsa.zhenzhen
cat ~/.ssh/id_rsa.zhenzhen.pub 保存到icode上
chmod 644 ~/.ssh/config
vim ~/.ssh/config
添加下自己的配置
Host icode.baidu.com
User yangyang22
IdentityFile ~/.ssh/id_rsa.yangyang22
（2）clone
git clone xxxxx代码库
git config --local user.email zhengzhenzhen@baidu.com
git config --local user.name zhengzhenzhen

git push origin master:refs/for/master
（3）上传文件
git add xxx
git commit -m '注释'
git push origin HEAD:refs/for/master

我不知道”undo”是什么意思。您可以删除core.excludesfile设置，如下所示：
git config --global --unset core.excludesfile
当然你也可以简单地编辑配置文件：
git config --global --edit
…然后手动删除设置。

git config --global --edit

git log
git reset --hard xxxxx

查看本地与云端的diff

git rm —cache 删除已经提交的内容
