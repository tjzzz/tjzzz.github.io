# java项目快速开发脚手架


## Guns
后台管理框架

采用主流框架 ： 基于 Spring Boot2.0+版本开发，并且支持 Spring Cloud Alibaba 微服务。
功能齐全 ：包含系统管理，代码生成，多数据库适配，SSO 单点登录，工作流，短信，邮件发送，OAuth2 登录，任务调度，持续集成，docker 部署等功能；
提供了企业版 ：企业版功能更多，并且提供了完善的开发文档，开发 demo，快速入门。
